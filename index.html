Übung 12

Auf der HTML Seite soll das Bild eines Roulette-Tisches angezeigt werden. (siehe Ordner "bilder")
Durch eine ImageMap soll reagiert werden, wenn der Benutzer auf eine einfache Chance klickt. https://www.ionos.at/digitalguide/websites/web-entwicklung/html-image-map/ https://summerstyle.github.io/summer/

Rot (rouge), Schwarz (noir), Gerade (pair), Ungerade (impair), manque (1-18), passe (19-36) Es soll die Funktion spielen() aufgerufen werden.
Die Funktion spielen() muss ebenfalls programiert werden.
Sie zieht eine Zufallszahl zwischen 0-36, gibt die Eigenschaften der Zahl aus und teilt dem Spieler mit, ob er gewonnen oder verloren hat. Bei 0 hat der Spieler weder gewonnen noch verloren.
Beispielausgaben: In diesem Beispiel klickt der Spieler jedes mal auf rot:
8, noir, pair, manque - Sie haben verloren
32, rouge, pair, passe - Sie haben gewonnen
0 - Ihr Einsatz bleibt liegen
